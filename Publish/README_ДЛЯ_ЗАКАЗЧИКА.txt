СОДЕРЖИМОЕ ДИСТРИБУТИВА:
-------------------------

1. AutoWorkShop.exe          - Исполняемый файл приложения
2. appsettings.json          - Файл конфигурации
3. 01_CreateDatabase_Full.sql - Скрипт создания и заполнения БД
4. 02_DatabaseCheck.sql      - Скрипт проверки БД
5. 03_DatabaseReset.sql      - Скрипт сброса БД
6. Deploy\                   - Папка с документами


ТРЕБОВАНИЯ К СИСТЕМЕ:
--------------------

1. Операционная система: Windows 10/11 (64-bit)
2. .NET 10.0 Runtime
3. SQL Server 2016+ или SQL Server Express


БЫСТРЫЙ СТАРТ:
--------------

ШАГ 1: Установка SQL Server
  - Скачайте SQL Server Express:
    https://www.microsoft.com/sql-server/sql-server-downloads
  - Установите, выберите "Базовая" установка

ШАГ 2: Создание базы данных
  - Откройте SQL Server Management Studio
  - Подключитесь к серверу
  - Откройте файл 01_CreateDatabase_Full.sql
  - Нажмите F5 (Выполнить)
  - Дождитесь сообщения "БАЗА ДАННЫХ УСПЕШНО СОЗДАНА!"

ШАГ 3: Настройка приложения
  - Откройте appsettings.json
  - При необходимости измените строку подключения:
    "Server=localhost\\SQLEXPRESS;Database=AutoWorkshopDB;..."

ШАГ 4: Запуск приложения
  - Запустите AutoWorkShop.exe
  - Войдите под admin / admin123
