# 🔧 АвтоМаст ИС
## Система управления автосервисом

**Версия:** 1.0  
**Дата:** Март 2026  
**Платформа:** Windows (.NET 10.0, WPF)

---

## 📖 ОПИСАНИЕ

**АвтоМаст ИС** — это информационная система для автоматизации работы автосервиса.  
Приложение позволяет управлять заказами, клиентами, сотрудниками, запчастями и финансами.

### Назначение системы

- Автоматизация бизнес-процессов автосервиса
- Учёт заказов и выполненных работ
- Управление складом запчастей
- Контроль финансовых операций
- Ведение базы клиентов и автомобилей
- Формирование отчётов и аналитики

---

## ✨ ВОЗМОЖНОСТИ СИСТЕМЫ

### 📋 Управление заказами
- Создание и редактирование заказов-нарядов
- Отслеживание статуса выполнения (Новый, В работе, Выполнен, Ожидает)
- Привязка запчастей и работ к заказу
- Автоматический расчёт стоимости
- История заказов по автомобилю

### 👥 Работа с клиентами
- База данных клиентов (физические и юридические лица)
- История обслуживаний по каждому клиенту
- Управление контактной информацией
- Быстрый поиск клиентов

### 🚗 Автопарк
- Учёт автомобилей клиентов
- VIN-номера и гос. номера
- Марка, модель, год выпуска
- История ремонтов по автомобилю

### 📦 Склад запчастей
- Учёт запчастей и материалов
- Контроль остатков на складе
- Минимальные пороги запаса
- Ценообразование и артикулы
- Движение запчастей по заказам

### 👨‍🔧 Сотрудники и отделы
- Организационная структура компании
- Справочник сотрудников (ФИО, должность, контакты)
- Привязка сотрудников к отделам
- Назначение ответственных за заказы

### 💰 Финансы
- Учёт оплат и поступлений
- Квитанции и чеки
- Финансовая отчётность
- Контроль дебиторской задолженности

### 📊 Статистика и аналитика
- Отчёты по заказам (по периодам, статусам, сотрудникам)
- Анализ доходов и прибыли
- Загрузка сотрудников и отделов
- Популярность услуг и запчастей

### 🔐 Система доступа
- **3 уровня прав:** Администратор, Мастер, Механик, Бухгалтер, Кладовщик
- Индивидуальные учётные записи
- Ведение журнала входов
- Блокировка пользователей

### 🔔 Уведомления
- Системные уведомления для пользователей
- Напоминания о заказах
- Оповещения о низких остатках запчастей

### 💾 Резервное копирование
- Экспорт данных в SQL-скрипт
- Сохранение резервных копий

## 🚀 БЫСТРЫЙ СТАРТ

### Для заказчиков

#### 1. Установка SQL Server
```
Скачайте и установите SQL Server Express:
https://www.microsoft.com/sql-server/sql-server-downloads
```

#### 2. Создание базы данных
```bash
# В SQL Server Management Studio выполните:
01_CreateDatabase_Full.sql
```

#### 3. Запуск приложения
```bash
# Запустите AutoWorkShop.exe
# Войдите под admin / admin123
```

Подробная инструкция: `Deploy\ИНСТРУКЦИЯ_ПО_УСТАНОВКЕ.md`

### Для разработчиков

```bash
# Сборка проекта
dotnet build --configuration Release

# Запуск
dotnet run

# Публикация
dotnet publish --configuration Release -r win-x64
```

## 📊 БАЗА ДАННЫХ

### Схема базы данных

```
AutoWorkshopDB
├── Departments          # Отделы компании
├── Employees            # Сотрудники
├── Clients              # Клиенты
├── Cars                 # Автомобили
├── Parts                # Запчасти
├── Orders               # Заказы
├── OrderParts           # Запчасти в заказах
├── Users                # Пользователи системы
├── Receipts             # Квитанции об оплате
├── Notifications        # Уведомления
└── Settings             # Настройки системы
```

### Основные таблицы

| Таблица | Описание | Ключевые поля |
|---------|----------|---------------|
| **Departments** | Отделы компании | Id, Name |
| **Employees** | Сотрудники | Id, FullName, Position, Phone, Email, DepartmentId |
| **Clients** | Клиенты | Id, FullName, Phone, Email, Address |
| **Cars** | Автомобили | Id, ClientId, Brand, Model, Year, VIN, PlateNumber |
| **Parts** | Запчасти | Id, Name, Article, Price, Quantity, MinQuantity |
| **Orders** | Заказы | Id, CarId, EmployeeId, Description, Status, TotalCost |
| **OrderParts** | Запчасти в заказе | Id, OrderId, PartId, Quantity, Price |
| **Users** | Пользователи | Id, Login, PasswordHash, Role, EmployeeId, IsActive |
| **Receipts** | Квитанции | Id, OrderId, Amount, PaymentDate, PaymentMethod |
| **Notifications** | Уведомления | Id, UserId, Message, IsRead, CreatedDate, Type |
| **Settings** | Настройки | Id, KeyName, Value, Description |

## 📦 РАЗВЁРТЫВАНИЕ

### Для заказчика

1. Установите SQL Server Express
2. Выполните скрипт `01_CreateDatabase_Full.sql`
3. Скопируйте файлы приложения
4. Настройте `appsettings.json`
5. Запустите `AutoWorkShop.exe`

См. `Deploy\ИНСТРУКЦИЯ_ПО_УСТАНОВКЕ.md`

### Для разработчика

```bash
# Сборка в Release режиме
dotnet build --configuration Release

# Публикация
dotnet publish --configuration Release -r win-x64 --self-contained false

# Копирование файлов
xcopy /E /I /Y bin\Release\net10.0-windows\publish\ C:\AutoWorkShop\
```

## 🔧 КОНФИГУРАЦИЯ

### appsettings.json

```json
{
    "ConnectionStrings": {
        "DefaultConnection": "Server=localhost;Database=AutoWorkshopDB;Trusted_Connection=True;TrustServerCertificate=True;"
    }
}
```

### Параметры подключения

| Параметр | Значение по умолчанию | Описание |
|----------|----------------------|----------|
| Server | localhost | Имя сервера SQL Server |
| Database | AutoWorkshopDB | Имя базы данных |
| Trusted_Connection | True | Использование Windows-аутентификации |
| TrustServerCertificate | True | Доверие сертификату сервера |
