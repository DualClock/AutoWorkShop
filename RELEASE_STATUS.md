## 📞 ИНСТРУКЦИЯ ДЛЯ ОТПРАВКИ

### Шаг 1: Запуск упаковки
```bash
package.bat
```

### Шаг 2: Создание архива
1. Найдите папку `AutoWorkShop_v1.0_2026-03-20`
2. Щёлкните правой кнопкой мыши
3. Выберите "Отправить" → "Сжатая ZIP-папка"

### Шаг 3: Отправка заказчику
Отправьте архив одним из способов:
- Email (если размер позволяет)
- Облачное хранилище (Google Disk, Yandex Disk, OneDrive)
- Файлообменник

### Шаг 4: Сопроводительное письмо
```
Уважаемый заказчик!

Направляем вам систему управления автосервисом "АвтоМаст ИС" версии 1.0.

В архиве:
- Программа для установки
- Инструкция по установке
- Скрипты для создания базы данных
- Учётные данные для входа

Требования к системе:
- Windows 10/11 (64-bit)
- SQL Server Express (бесплатно)
- .NET 10.0 Runtime

Подробная инструкция в файле НАЧНИТЕ_ЗДЕСЬ.txt

С уважением,
Разработчик
```
